# py_isx

Experimental pure-python API to read ISXD files

## Support

|  File type | Support |
|  --------- | ------- |
| CellSet   | ✅ |
| Movie   | ✅ |
| VesselSet   | ✅ |
| Events   | ❌ |


## Installation

### Poetry

```bash
poetry add https://github.com/inscopix/py_isx
```

### pip


```bash
pip install https://github.com/inscopix/py_isx
```
